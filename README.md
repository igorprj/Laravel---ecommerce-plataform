# Laravel E-commerce Platform (Car Dealership Example)

## Overview
This is a full-stack e-commerce system built with Laravel and Bootstrap. 
The project focuses on backend architecture, authentication, and database management.

## Features
- User authentication system
- Access control (only logged users can enter the system)
- Product listing with image, description, and price
- Search functionality
- Admin panel to manage products
- Database integration

## Technologies
- PHP
- Laravel
- Bootstrap
- MySQL

## How to Run
1. Copy `.env.example` to `.env`
2. Configure your database in `.env`
3. Run `composer install`
4. Run `php artisan migrate`
5. Run `php artisan serve`
6. Open the app in your browser
